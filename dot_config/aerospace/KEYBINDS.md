# AeroSpace キーバインド一覧

## 基本情報

- 修飾キー: `Alt` (メインモード)
- サービスモード: `Alt+Shift+;` で切り替え

## レイアウト操作

| キー         | 説明                                           |
| ------------ | ---------------------------------------------- |
| `Alt+/`      | タイルレイアウト（横/縦を切り替え）            |
| `Alt+,`      | アコーディオンレイアウト（横/縦を切り替え）    |

## ウィンドウフォーカス

| キー     | 説明               |
| -------- | ------------------ |
| `Alt+h`  | 左のウィンドウ     |
| `Alt+j`  | 下のウィンドウ     |
| `Alt+k`  | 上のウィンドウ     |
| `Alt+l`  | 右のウィンドウ     |

## ウィンドウ移動

| キー           | 説明                   |
| -------------- | ---------------------- |
| `Alt+Shift+h`  | ウィンドウを左に移動   |
| `Alt+Shift+j`  | ウィンドウを下に移動   |
| `Alt+Shift+k`  | ウィンドウを上に移動   |
| `Alt+Shift+l`  | ウィンドウを右に移動   |

## ウィンドウリサイズ

| キー     | 説明                 |
| -------- | -------------------- |
| `Alt+-`  | ウィンドウを縮小     |
| `Alt+=`  | ウィンドウを拡大     |

## サービスモード (`Alt+Shift+;` で入る)

サービスモードに入った後、以下のキーが使えます。

| キー                | 説明                                            |
| ------------------- | ----------------------------------------------- |
| `Esc`               | 設定を再読み込みしてメインモードに戻る          |
| `r`                 | レイアウトをリセットしてメインモードに戻る      |
| `f`                 | フローティング/タイリング切り替え               |
| `Backspace`         | 現在のウィンドウ以外をすべて閉じる              |
| `Alt+Shift+h`       | 左のウィンドウと結合してメインモードに戻る      |
| `Alt+Shift+j`       | 下のウィンドウと結合してメインモードに戻る      |
| `Alt+Shift+k`       | 上のウィンドウと結合してメインモードに戻る      |
| `Alt+Shift+l`       | 右のウィンドウと結合してメインモードに戻る      |

## キー修飾キーの説明

- `Alt`: Option キー (macOS)
- `Shift`: Shift キー
- `Tab`: Tab キー

## その他の機能

- マウスは自動的にフォーカスされたモニターの中央に移動します
- ワークスペース 1-9 および A-Z は常に存在します（persistent-workspaces）
- ウィンドウ間のギャップは 10px に設定されています
