# wezterm キーバインド一覧

リーダーキー: `Ctrl+q` (2 秒以内に次のキーを押す)

## ワークスペース操作

| キー               | 説明                                   |
| ------------------ | -------------------------------------- |
| `Ctrl+q` `w`       | ワークスペース一覧を表示して切り替え   |
| `Ctrl+q` `$`       | 現在のワークスペースの名前を変更       |
| `Ctrl+q` `Shift+W` | 新しいワークスペースを作成して切り替え |

## タブ操作

| キー             | 説明                           |
| ---------------- | ------------------------------ |
| `Cmd+t`          | 新しいタブを作成               |
| `Cmd+w`          | 現在のタブを閉じる（確認あり） |
| `Ctrl+Tab`       | 次のタブに移動                 |
| `Ctrl+Shift+Tab` | 前のタブに移動                 |
| `Ctrl+q` `{`     | タブを左に移動                 |
| `Ctrl+q` `}`     | タブを右に移動                 |
| `Cmd+1`          | タブ 1 に移動                  |
| `Cmd+2`          | タブ 2 に移動                  |
| `Cmd+3`          | タブ 3 に移動                  |
| `Cmd+4`          | タブ 4 に移動                  |
| `Cmd+5`          | タブ 5 に移動                  |
| `Cmd+6`          | タブ 6 に移動                  |
| `Cmd+7`          | タブ 7 に移動                  |
| `Cmd+8`          | タブ 8 に移動                  |
| `Cmd+9`          | 最後のタブに移動               |

## ペイン操作

| キー           | 説明                               |
| -------------- | ---------------------------------- |
| `Ctrl+q` `\`   | ペインを縦に分割                   |
| `Ctrl+q` `-`   | ペインを横に分割                   |
| `Ctrl+q` `x`   | 現在のペインを閉じる（確認あり）   |
| `Ctrl+q` `h`   | 左のペインに移動                   |
| `Ctrl+q` `l`   | 右のペインに移動                   |
| `Ctrl+q` `k`   | 上のペインに移動                   |
| `Ctrl+q` `j`   | 下のペインに移動                   |
| `Ctrl+q` `z`   | 現在のペインをズーム（全画面表示） |
| `Ctrl+Shift+[` | ペイン選択モード                   |
| `Ctrl+q` `s`   | ペインサイズ調整モード（後述）     |
| `Ctrl+q` `a`   | ペイン移動モード（1 秒間、後述）   |

### ペインサイズ調整モード (`Ctrl+q` `s`)

| キー    | 説明                |
| ------- | ------------------- |
| `h`     | 左に 1 ピクセル拡大 |
| `l`     | 右に 1 ピクセル拡大 |
| `k`     | 上に 1 ピクセル拡大 |
| `j`     | 下に 1 ピクセル拡大 |
| `Enter` | モードを終了        |

### ペイン移動モード (`Ctrl+q` `a`)

1 秒間有効。この間に以下のキーでペインを移動できます。

| キー | 説明             |
| ---- | ---------------- |
| `h`  | 左のペインに移動 |
| `l`  | 右のペインに移動 |
| `k`  | 上のペインに移動 |
| `j`  | 下のペインに移動 |

## コピー・貼り付け

| キー         | 説明                             |
| ------------ | -------------------------------- |
| `Cmd+c`      | 選択範囲をクリップボードにコピー |
| `Cmd+v`      | クリップボードから貼り付け       |
| `Ctrl+q` `[` | コピーモードを開始（後述）       |

### コピーモード (`Ctrl+q` `[`)

vi 風のキーバインドでスクロールバックを操作できます。

#### 基本移動

| キー            | 説明                         |
| --------------- | ---------------------------- |
| `h` `j` `k` `l` | 左・下・上・右に移動         |
| `0`             | 行の先頭に移動               |
| `^`             | 行の最初の文字に移動         |
| `$`             | 行の最後に移動               |
| `g`             | スクロールバックの先頭に移動 |
| `G`             | スクロールバックの最後に移動 |

#### 単語移動

| キー | 説明                 |
| ---- | -------------------- |
| `w`  | 次の単語の先頭に移動 |
| `b`  | 前の単語の先頭に移動 |
| `e`  | 次の単語の末尾に移動 |

#### ジャンプ機能

| キー       | 説明                                     |
| ---------- | ---------------------------------------- |
| `f` + 文字 | 次の指定文字までジャンプ（含む）         |
| `F` + 文字 | 前の指定文字までジャンプ（含む）         |
| `t` + 文字 | 次の指定文字の前までジャンプ（含まない） |
| `T` + 文字 | 前の指定文字の後までジャンプ（含まない） |
| `;`        | 前回のジャンプを繰り返し                 |

#### ビューポート移動

| キー | 説明                     |
| ---- | ------------------------ |
| `H`  | ビューポートの上端に移動 |
| `M`  | ビューポートの中央に移動 |
| `L`  | ビューポートの下端に移動 |

#### スクロール

| キー     | 説明                   |
| -------- | ---------------------- |
| `Ctrl+b` | 1 ページ上にスクロール |
| `Ctrl+f` | 1 ページ下にスクロール |
| `Ctrl+u` | 半ページ上にスクロール |
| `Ctrl+d` | 半ページ下にスクロール |

#### 選択モード

| キー      | 説明                               |
| --------- | ---------------------------------- |
| `v`       | セル選択モード                     |
| `Ctrl+v`  | ブロック選択モード                 |
| `Shift+v` | 行選択モード                       |
| `o`       | 選択範囲の反対側の端に移動         |
| `Shift+O` | 選択範囲の反対側の端に移動（水平） |

#### コピー・終了

| キー     | 説明                             |
| -------- | -------------------------------- |
| `y`      | 選択範囲をクリップボードにコピー |
| `Enter`  | コピーしてモードを終了           |
| `Escape` | モードを終了（コピーしない）     |
| `Ctrl+c` | モードを終了（コピーしない）     |
| `q`      | モードを終了（コピーしない）     |

## フォントサイズ

| キー     | 説明                     |
| -------- | ------------------------ |
| `Ctrl++` | フォントサイズを大きく   |
| `Ctrl+-` | フォントサイズを小さく   |
| `Ctrl+0` | フォントサイズをリセット |

## ウィンドウ操作

| キー        | 説明                   |
| ----------- | ---------------------- |
| `Cmd+n`     | 新しいウィンドウを開く |
| `Alt+Enter` | フルスクリーン切り替え |

## その他

| キー           | 説明                     |
| -------------- | ------------------------ |
| `Cmd+p`        | コマンドパレットを表示   |
| `Ctrl+Shift+p` | コマンドパレットを表示   |
| `Ctrl+Shift+r` | 設定ファイルを再読み込み |

## キー修飾キーの説明

- `LEADER`: `Ctrl+q` (リーダーキー、2 秒以内に次のキーを押す)
- `SUPER`: `Cmd` (macOS)
- `CTRL`: `Ctrl`
- `SHIFT`: `Shift`
- `ALT`: `Alt`
