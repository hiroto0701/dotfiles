# Yazi キーバインド一覧

## ファイルマネージャー

### 基本操作

| キー | 動作 |
|---|---|
| `q` | 終了 |
| `Q` | 終了（cwd-file に書き込まない） |
| `<Esc>` | ビジュアルモード終了 / 選択解除 / 検索キャンセル |
| `<C-c>` | 現在のタブを閉じる（最後のタブなら終了） |

### ナビゲーション

| キー | 動作 |
|---|---|
| `h` | 親ディレクトリへ |
| `l` | 子ディレクトリへ |
| `j` | 下へ移動 |
| `k` | 上へ移動 |
| `J` | 5行下へ移動 |
| `K` | 5行上へ移動 |
| `H` | 履歴：前のディレクトリへ |
| `L` | 履歴：次のディレクトリへ |
| `<C-u>` | 半ページ上へ |
| `<C-d>` | 半ページ下へ |
| `<C-b>` | 1ページ上へ |
| `<C-f>` | 1ページ下へ |
| `g` | 先頭へ移動 |
| `G` | 末尾へ移動 |

### ファイル操作

| キー | 動作 |
|---|---|
| `o` / `<Enter>` | ファイルを開く |
| `O` | インタラクティブに開く（nvim / Cursor / VSCode を選択） |
| `y` | コピー（yank） |
| `x` | カット |
| `p` | ペースト |
| `P` | 強制ペースト（上書き） |
| `d` | ゴミ箱へ移動 |
| `D` | 完全削除 |
| `a` | 作成（`/` で終わるとディレクトリ） |
| `r` | リネーム |
| `u` | undo |
| `<C-r>` | redo |
| `.` | 隠しファイルの表示切替 |

### 検索・フィルター

| キー | 動作 |
|---|---|
| `f` | フィルター |
| `/` | 前方検索 |
| `?` | 後方検索 |
| `n` | 次の検索結果へ |
| `N` | 前の検索結果へ |
| `s` | ファイル名で検索（fd） |
| `S` | ファイル内容で検索（ripgrep） |
| `<C-s>` | 検索キャンセル |
| `z` | zoxide でディレクトリジャンプ |
| `Z` | fzf でディレクトリジャンプ |

### 選択

| キー | 動作 |
|---|---|
| `v` | ビジュアルモード（選択モード） |
| `V` | ビジュアルモード（解除モード） |
| `<C-a>` | 全選択 |
| `<C-r>` | 選択反転 |

### パスのコピー

| キー | 動作 |
|---|---|
| `cc` | ファイル名（フルネーム） |
| `cn` | ファイル名（拡張子なし） |
| `ce` | 拡張子のみ |
| `cp` | 絶対パス |
| `cd` | 親ディレクトリのパス |

### ソート

| キー | 動作 |
|---|---|
| `,m` | 更新日時でソート |
| `,M` | 更新日時でソート（逆順） |
| `,s` | サイズでソート |
| `,S` | サイズでソート（逆順） |
| `,n` | 自然順でソート |
| `,e` | 拡張子でソート |

### 表示モード（linemode）

| キー | 動作 |
|---|---|
| `ms` | サイズを表示 |
| `mp` | パーミッションを表示 |
| `mm` | 更新日時を表示 |
| `mn` | 非表示 |

### タブ

| キー | 動作 |
|---|---|
| `t` | 現在のパスで新タブ |
| `1〜5` | 対応するタブへ切替 |
| `<Tab>` | 次のタブへ |
| `<S-Tab>` | 前のタブへ |
| `[` | 前のタブと入れ替え |
| `]` | 次のタブと入れ替え |

### ジャンプ（goto）

| キー | 動作 |
|---|---|
| `gh` | ホームディレクトリ（`~`） |
| `gd` | `~/Downloads` |
| `gc` | `~/.config` |
| `gt` | `/tmp` |

### シェル

| キー | 動作 |
|---|---|
| `!` | シェルコマンド実行（ブロック） |
| `\|` | シェルコマンド実行（非ブロック） |
| `<Space>g` | lazygit を起動 |

## その他の UI

### タスクマネージャー

| キー | 動作 |
|---|---|
| `j` / `k` | 上下移動 |
| `<Enter>` | タスクの詳細を表示 |
| `x` | タスクをキャンセル |
| `<Esc>` / `<C-c>` | 閉じる |

### 選択ダイアログ

| キー | 動作 |
|---|---|
| `j` / `k` | 上下移動 |
| `<Enter>` | 選択を確定 |
| `<Esc>` / `<C-c>` | キャンセル |

### 入力ダイアログ

| キー | 動作 |
|---|---|
| `i` / `a` | 挿入モード / 追記モード |
| `h` / `l` | 1文字左右移動 |
| `b` / `w` | 1単語左右移動 |
| `<C-a>` / `<C-e>` | 行頭 / 行末へ |
| `<C-u>` / `<C-k>` | 行頭まで削除 / 行末まで削除 |
| `<C-w>` / `<C-d>` | 前の単語を削除 / 次の単語を削除 |
| `<C-v>` | クリップボードからペースト |
| `<Enter>` | 確定 |
| `<Esc>` | ノーマルモードへ / キャンセル |

### 確認ダイアログ

| キー | 動作 |
|---|---|
| `y` | 確定 |
| `n` | キャンセル |
| `<Enter>` | 確定 |
| `<Esc>` / `<C-c>` | キャンセル |

### ヘルプ

| キー | 動作 |
|---|---|
| `j` / `k` | 上下移動 |
| `/` | フィルター |
| `<Esc>` / `<C-c>` | 閉じる |
